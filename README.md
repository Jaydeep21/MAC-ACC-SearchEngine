## Install source code

1. Dependencies

* openjdk >= 8
* maven > 3

2. Download source code

```
$ git clone https://github.com/Jaydeep21/MAC-ACC-SearchEngine.git
$ cd MAC-ACC-SearchEngine
```

**Build and Run**

```
$ cd MAC-ACC-SearchEngine
$ mvn clean install
### Run WebSearchEngine.java 
```
